# REV-001: CI Rust TSan ABI Mismatch

## Type
CI/DevOps — GitHub Actions YAML

## Problem
`sanitizer-tsan` job in `.github/workflows/ci-rust-10.yml` uses `-Cunsafe-allow-abi-mismatch=sanitizer` flag that is incompatible with the current nightly Rust toolchain. This breaks CI on `main` and blocks releases.

## Solution
Remove `-Zsanitizer=thread` and `-Cunsafe-allow-abi-mismatch=sanitizer` from the TSan job's `RUSTFLAGS` and `RUSTDOCFLAGS`. The job continues to run `cargo +nightly test` which still provides nightly regression detection value.

## Contract
- `cargo nextest run --profile audit --workspace --build-jobs 2` passes in CI
- YAML is syntactically valid

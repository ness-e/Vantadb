# IQL Moved To Experimental

IQL is not part of the v0.1.x stable API surface. The historical specification
now lives at [../experimental/IQL.md](../experimental/IQL.md).

The stable MVP API remains the embedded SDK/CLI memory surface: CRUD, namespace
retrieval, vector search, BM25 text search, Hybrid Retrieval v1,
rebuild/audit/repair, and JSONL export/import.

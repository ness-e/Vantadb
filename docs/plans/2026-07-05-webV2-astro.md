# webV2 — VantaDB Website Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Rebuild VantaDB's marketing website as `webV2/` using Astro 5, migrating from the current Vite+React SPA. 0 JS by default, GSAP-powered 2D animations, Swiss/Neubrutalism design language, Markdown blog.

**Architecture:** Astro 5 static site with React islands for interactive/animated sections (Hero, Stats, Features, Showcase, HowItWorks). Content Collections for blog. TailwindCSS v4 for styling. GSAP 3 + ScrollTrigger for animations. No images — all visuals are CSS procedural textures (grid, scanlines, noise, dot patterns) + typography.

**Tech Stack:**
- Astro 5, React 19 (islands), TypeScript
- TailwindCSS v4, GSAP 3.15, @gsap/react
- @fontsource-variable/space-grotesk, outfit, jetbrains-mono
- Playwright (for product screenshots in Showcase)

**Design System:** Swiss + Neubrutalism — dark bg (#0d0d0d), white text (#f5f5f5), amber accent (#ff5500), 0 border-radius, hard offset shadows, terminal aesthetic.

## Brand References

Key design rules (from `docs/web/DESIGN.md`):
- **95/5 color rule**: 95% black/white/gray, 5% amber accent
- **Zero border-radius** on anything
- **Hard offset shadows** (`Xpx Ypx 0px 0px`, no blur)
- **Terminal aesthetic**: `>` prefixes, `[ ]` brackets, ALL CAPS mono labels
- **Mechanical interactions**: buttons press down (shadow reduces on hover/active)
- **No gradients, no glassmorphism, no illustrations, no photos**
- **Fonts**: Space Grotesk (display), Outfit (body), JetBrains Mono (code/labels)
- **Animations**: 50-150ms, --ease-brutal, no bounce/elastic
